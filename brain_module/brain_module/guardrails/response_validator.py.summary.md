# File: response_validator.py

## Purpose
Validate the response generated by a language model to ensure it meets certain criteria.

## Key Components
- Four heuristic checks:
  1. Negative lead detection
  2. Self-contradiction scan
  3. Answer-question alignment
  4. Confidence gate
- Customizable parameters for confidence thresholds and strict mode

## Important Logic
The `validate_response` function runs all four checks on the answer and returns a `ValidationResult` object with any issues found and a suggested action.

## Dependencies
- `logging`: for logging warning messages when guardrail issues are detected
- `re`: for regular expression matching in negative lead detection and self-contradiction scan

## Notes
This code provides a flexible way to validate responses from language models, allowing for customization of confidence thresholds and strict mode. The checks are lightweight and do not require additional LLM calls.
