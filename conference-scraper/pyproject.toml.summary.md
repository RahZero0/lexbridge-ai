# File: pyproject.toml

## Purpose
Define the project's metadata and dependencies for use in package managers like pip.

## Key Components
- **Project Information**: `name`, `version`, `description` and `readme` fields specify basic details about the project.
- **Python Version Requirement**: The `requires-python` field specifies that this project requires a Python version of at least 3.14 to run.
- **Dependencies**: Lists the external libraries required for the project, including `beautifulsoup4`, `playwright`, and `requests`.
